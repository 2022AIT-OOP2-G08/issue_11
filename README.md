# グループ 8 第 11 回課題

Web インターフェイスを持つ画像処理システムの構築

# Requirement

- opencv 4.6.0.66
- flask 2.2.2
- watchdog 2.2.0

# Installation

```bash
pip install python-opencv
pip install flask
pip install watchdog
```

# Usage

```bash
$ git clone https://github.com/2022AIT-OOP2-G08/issue_11.git
$ cd issue_11
$ python main.py
$ python modules/observation.py
```
